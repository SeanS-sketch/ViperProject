Project Overview

Viper Rover is being built through an 8-milestone development roadmap for learning robot programming with the ELEGOO Smart Robot Car 4.0. The current focus is the Block Workspace, with Milestone 3 completed as a visual editor for arranging blocks. Future milestones will add richer block types, an execution engine, and Bluetooth control through the Android app.

Status

  • Milestone 1: Project foundation Complete
  • Milestone 2: Bluetooth connection layer Complete
  • Milestone 3: Visual Block Workspace Complete
  • Milestones 4-8: Blocks, execution, and rover control In progress / locked
